## Deployment

### Podman Compose

```yaml
services:
  tailscale:
    image: ghcr.io/daemonless/tailscale:latest
    container_name: tailscale
    environment:
      - TS_AUTHKEY=tskey-auth-xxxx
      - TS_EXTRA_ARGS=--advertise-exit-node
    volumes:
      - "/path/to/containers/tailscale:/config"
    restart: unless-stopped
```
